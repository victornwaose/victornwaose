### 🧑‍💻 About Me

I'm a passionate frontend developer who obsesses over clean UI, smooth interactions, and accessible design. I'm now expanding into full-stack development so I can own entire products — from database schema to deployed interface — and build things that genuinely improve people's lives.

I started coding because I wanted to solve real problems I saw around me. That drive hasn't changed. Whether it's a small tool that saves someone an hour a day, or a platform that connects an entire community — I want every project to matter.

- 🎨 Specialise in **Vue.js**, **React**, **Nuxt.js**, and **Next.js**
- 📦 Write clean, scalable code with **TypeScript** and **TailwindCSS**
- ♿ Design with **accessibility** in mind — every user deserves a great experience
- 🌱 Currently learning **Node.js**, **Laravel**, **REST APIs**, and **MySQL**
- 🤝 Love collaborating on **open-source** projects that make a real impact
- 💬 Ask me about **frontend architecture**, **component design**, or **UI/UX decisions**
- 🎯 Goal: Ship my first full-stack SaaS product in 2026
- 🌍 Based in **Asaba, Delta State, Nigeria** — working globally
- ⚡ Fun fact: I believe the best UIs are the ones users never have to think about
